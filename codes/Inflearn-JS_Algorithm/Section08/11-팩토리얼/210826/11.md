## 인프런 - 자바스크립트 알고리즘 풀이

### **11.** 재귀함수와 완전탐색(깊이우선탐색, DFS) - 팩토리얼

**> 문제**

자연수 N을 입력하면 N!값을 구하세요.  
N! = n*(n-1)*(n-2)*.....*2*1입니다.  
만약 N=5라면 5!=5*4*3*2\*1=120입니다.

- 입력: 5
- 출력: 120

**> 코드**

_ME_

```js
function solution(n) {
  if (n <= 1) return n;
  let result = n;
  function DFS(L) {
    if (L === 0) return;
    result *= L;
    DFS(L - 1);
  }
  DFS(n - 1);
  return result;
}

console.log(solution(5));
```

- 이 강의에선 DFS를 결과 값을 나타내는 변수에 대입을 해서 풀었어야하는데,  
   그게 아닌 return 값이 없는 함수를 작성해서 해결을 했음.
  - 아래 (_ANSWER_)에 흘러가는 흐름 메모하며 이해

_ANSWER_

```js
function solution(n) {
  let answer;
  function DFS(n) {
    if (n === 1) return 1;
    else return n * DFS(n - 1);
  }
  answer = DFS(n);
  return answer;
}

console.log(solution(5));
```

이 코드의 동작 과정

- `answer = DFS(n)`: DFS 내부에서 다시 재귀하는 DFS까지 계산한 후 대입 됨.
- n은 5, n이 1이 될때까지 DFS는 재귀 됨
- 콜스택에 들어가는 순서  
   DFS(5) -> DFS(4) -> DFS(3) -> DFS(2) -> DFS(1)
- 콜스택에서 호출 시  
   DFS(1): 1 ->  
   DFS(2): 2(1 \* 2) ->  
   DFS(3) : 6 (1 \* 2 \* 3) ->  
   DFS(4) : 24 (1 \* 2 \* 3 \* 4) ->  
   DFS(5) : 120 (1 \* 2 \* 3 \* 4 \* 5)
