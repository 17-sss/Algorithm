## 인프런 - 자바스크립트 알고리즘 풀이

### **04.** 기본 문제 풀이 - 1부터 N까지의 합

**> 문제**

자연수 N이 입력되면 1부터 N까지의 합을 출력하는 프로그램을 작성.

**입출력 - 1**

-   입력 1: 6
-   출력 1: 21

**입출력 - 2**

-   입력 2: 10
-   출력 2: 55

**> 코드**

```js
const solution = (n) => {
    let answer = 0;
    for (let i = 1; i <= n; i++) answer += i;
    return answer;
}
```
